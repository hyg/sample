# E2EE MQTT Chat - Web 客户端（本地文件版）

## 目录结构

```
web-local/
├── index.html              # 主页面（使用本地依赖）
├── e2ee/
│   └── hpke-browser.js     # HPKE 加密模块
├── lib/                    # 本地依赖库
│   ├── curves/             # @noble/curves
│   ├── hashes/             # @noble/hashes
│   └── mqtt.esm.js         # MQTT 客户端
├── README.md               # 本文件
└── DEPLOY.md               # 部署指南
```

## 特点

- **完全离线**：所有依赖都在本地，不需要 CDN
- **部署简单**：只需上传整个文件夹
- **适合内网**：可以在没有外网连接的环境使用

## 部署

将整个 `web-local/` 文件夹上传到服务器即可。

**注意**：
- 需要 HTTPS 支持（WSS 需要）
- 文件大小约 2-3 MB（包含所有依赖）

## 使用方法

1. **访问页面**：打开浏览器访问 `https://your-domain.com`

2. **创建身份**：
   - 点击"创建 X25519 身份"
   - 保存显示的 DID 和公钥（十六进制）

3. **交换信息**：
   - 将你的 DID 和公钥发送给通信伙伴
   - 获取伙伴的 DID 和公钥

4. **连接伙伴**：
   - 在"伙伴的 DID"输入框粘贴伙伴的 DID
   - 在"伙伴的公钥"输入框粘贴伙伴的公钥（十六进制）
   - 点击"连接"

5. **初始化加密会话**：
   - 点击"初始化 E2EE 会话"
   - 等待会话建立

6. **开始聊天**：
   - 在输入框输入消息
   - 按 Enter 或点击"发送"
